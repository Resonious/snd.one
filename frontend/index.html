snd.one

Suite of HTTP-based tools for programmers. Inspired by request bin and the likes.

No installs, no sign-ups. Just use them if they seem helpful.

Inspect live webhooks via terminal

Direct your webhooks to https://hook.snd.one/anything.

Curl the same URL to get a live stream of incoming requests.

curl -L hook.snd.one/can-be-anything Your URL is https://hook.snd.one/can-be-anything POST /can-be-anything host: "hook.snd.one" user-agent: "curl/8.2.1" accept: "*/*" content-length: "9" content-type: "application/x-www-form-urlencoded" some data
curl -Ld 'some data' hook.snd.one/can-be-anything

Receive desktop/mobile notifications via HTTP request

Use this service to, for example, get a notification when a long CI job finishes.

What is all this?

Right now it's a hobby project, but it solves real problems that I have during my day job as a web programmer. Many similar services already exist, but most of them are complicated or slow to the point where I'm better off writing a custom script every time. These snd.one utilities aim to be much simpler and much faster.
